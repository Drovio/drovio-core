Relations person phone manager

Every relation person can have 0 or more phones and they can by of different type (home, work, mobile, fax etc.). You can add phones to a person as easy as two lines of code and you can edit and remove them even easier:

// Use classes
use \ENP\Relations\ePerson;
use \ENP\Relations\ePersonPhone;

// Get person id
$personID = engine::getVar("id");

// Create an empty phone instance
$pPhone = new ePersonPhone($personID);

// Create new phone (mobile, fax, work, home etc.)
$pPhone->create($typeID = 1, $phone = "6931234567", $countryID = "");


// Update the phone info
$pPhone->update($typeID = 1, $phone = "6931234568", $countryID = "87");

Select a phone type

When creating or updating a phone, we need to provide a phone type. This class can provide us with a list of all phone types that the SDK supports:

// Use classes
use \ENP\Relations\ePersonPhone;

// Get all phone types
$types = new ePersonPhone::getPhoneTypes();

// The above should return something like this:
// It is: key => title
Array
(
    [0] => "Mobile"
    [1] => "Home"
    [2] => "Fax"
    [3] => "Work fax"
    [4] => "Other"
)
