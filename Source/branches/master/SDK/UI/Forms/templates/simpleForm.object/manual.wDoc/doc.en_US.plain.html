Simple Form Template

Using the Form builder/factory, we have created a simple form template where a developer can easily create a complete form in just a few lines of code having all the features and the behavior a form should have.

This template includes pairing labels with inputs, make required fields more visible, inserting notes for extended description and ready action controls.

Lets use the simple form template to nice form:

// Create a form, build and append items and inputs
	
// Use the form object
use \UI\Forms\templates\simpleForm; // extends Form
	
// Create a form object
$form = new simpleForm();

// This is a UIObject (extends formItem), so we build
// We use the chain pattern to get the DOMElement immediately
// It supports multipart forms with a single parameter
// We also set to have default buttons, execute and reset
$threadForm = $form->build($action = "/forum/createThread.php", $defaultButtons = TRUE, $async = TRUE, $fileUpload = FALSE)->get();

// Append form to form container
DOM::append($formContainer, $profileForm);

// Add a hidden input of the forum id
$input = $form->getInput($type = "hidden", $name = "fid", $value = $forumID, $class = "", $autofocus = FALSE, $required = FALSE);
$form->append($input);

// Thread name
// Build a form row with label and input
$input = $form->getInput($type = "text", $name = "name", $value = "", $class = "", $autofocus = FALSE, $required = TRUE);
$frow = $form->buildRow("Thread name", $input, $required = TRUE, $notes = "");
$form->append($frow);

// Thready body
// Build a form row with label and textarea
$input = $form->getTextarea($name = "body", $value = "", $class = "");
$frow = $form->buildRow("Thread body", $input, $required = FALSE, $notes = "");
$form->append($frow);

On the above example we see the logic behind the creation of a simple form. We can extend those features and add any input or form item on the form any way we want.

Javascript interface + examples

Simple Form template also provides a javascript interface where a developer can create an entire form with its inputs on the client side dynamically.

// Build a row like before
Function:	buildRow()
Description:	Build a form row including a label and an input
Usage:		simpleForm.buildRow(form, title, input, required, notes);
Example:
	// Get the form
	var jqForm = jq("form.userForm");
	
	// We can build a row, customize it and then add it to the form
	// Create form input
	var input = Form.getInput(jqForm, "text", "lastname", "", "", true, true);
	
	// Build the form row
	var frow = simpleForm.buildRow(jqForm, "Last Name", input, true, "");
	
	// Append the row to the form
	jqForm.append(frow);



Function:	insertRow()
Description:	Insert a ready row to the form
Usage:		simpleForm.insertRow(form, title, input, required, notes);
Example:
	// Get the form
	var jqForm = jq("form.userForm");
	
	// Create form input
	var input = Form.getInput(jqForm, "text", "firstname", "", "", true, true);
	
	// Create and insert the form row
	simpleForm.insertRow(jqForm, "First Name", input, true, "");



Function:	getRow()
Description:	Get a row element
Usage:		simpleForm.getRow();

Function:	getSimpleLabel()
Description:	Get a label customized for this template
Usage:		simpleForm.getSimpleLabel(form, title, forInputID);

Function:	getTextarea()
Description:	Get a form textarea
Usage:		Form.getTextarea(form, name, value, areaClass, autofocus);

Function:	getNotes()
Description:	Get form row notes to describe the input
Usage:		simpleForm.getNotes(notes);
