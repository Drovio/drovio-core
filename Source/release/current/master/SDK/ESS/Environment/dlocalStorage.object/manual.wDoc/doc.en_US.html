Local Storage manager

An easy way to manage local and session storage.

Javascript interface

// Javascript interface
	
// Set a value
Function:	set()
Description:	Set a localStorage or sessionStorage value. It will transform almost any object to string. Set st_session to true to set sessionStorage, if not set it will be considered false.
Usage:		dlocalStorage.set(st_name, st_value, st_session);
Example:	dlocalStorage.set("storage_var_name", "storage_a_value", false);

// Get a value from local or session storage
Function:	get()
Description:	Get a value from the storage. Set st_session to true to get from sessionStorage, if not set it will be considered false.
Usage:		dlocalStorage.get(st_name, st_session);
Example:	dlocalStorage.get("storage_var_name", false);

// Remove a value from local or session storage
Function:	remove()
Description:	Remove a value from the storage. Set st_session to true to remove from sessionStorage, if not set it will be considered false.
Usage:		dlocalStorage.remove(st_name, st_session);
Example:	dlocalStorage.remove("storage_var_name", false);
