Mailgun interface

You can use this interface to send emails with a mailgun api key from any domain. We don't store any information anywhere.

This class is provided to save some time and instead of doing the full API requests, we do it for you.

Sending a simple email

Mailgun provides the simplest interface for sending an email, and we are passing it through to the entire SDK:

// Use the class
use \API\Comm\mail\mailgun;

// Create a mailgun instance with the desired domain
// We are using version 3, the default one
$mailgun = new mailgun($domain = "test.domain", $apikey = "api:key-02517816282e426db57914d4b68d2437", $version = 3);

// Set the sender address
$mailgun->setFrom($from = "my_email@test.domain");

// Add a recipient
$mailgun->addRecipient($address = "customer@company.company_domain", $type = self::RCP_TO);

// Send the email
// It supports both text and html content
$mailgun->send($subject = "Test email", $textContent = "This is a test email", $htmlContent = "
This is a test email
");

Attachments

The mailgun API allows us to add attachments to the email. This class provides direct access to the team directory and you can attach a file directly by giving the inner path.

// Use the class
use \API\Comm\mail\mailgun;

// Create a mailgun instance
$mailgun = new mailgun($domain = "test.domain", $apikey = "api:key-02517816282e426db57914d4b68d2437", $version = 3);
$mailgun->setFrom($from = "my_email@test.domain");

// Add attachment given the file path
$filePath = "/path/to/file";
$mailgun->addAttachment($filePath);

// Send the email with the attachment
$mailgun->send($subject = "Test email", $textContent = "This is a test email", $htmlContent = "
This is a test email
");
