Relations person address manager

Every relation person can have 0 or more addresses and they can by of different type (home, work etc.). You can add addresses to a person as easy as two lines of code and you can edit and remove them even easier:

// Use classes
use \ENP\Relations\ePerson;
use \ENP\Relations\ePersonAddress;

// Get person id
$personID = engine::getVar("id");

// Create an empty address instance
$pAddress = new ePersonAddress($personID);

// Create new address
$pAddress->create($typeID, $address = "Egnatia", $postal_code = "54621", $city = "Thessaloniki", $countryID = "");


// Update the address info
$pAddress->update($typeID, $address = "Egnatia road", $postal_code = "54621", $city = "Thessaloniki", $countryID = "87");

Select an address type

When creating or updating an address, we need to provide an address type. This class can provide us with a list of all address types that the SDK supports:

// Use classes
use \ENP\Relations\ePersonAddress;

// Get all address types
$types = new ePersonAddress::getAddressTypes();

// The above should return something like this:
// It is: key => title
Array
(
    [0] => "Home"
    [1] => "Work"
    [2] => "Fax"
)
