Retail product manager

This is the basic class for managing retail products. It provide an interface to the Retail SDK for creating, updating and removing products for the current company (extends team).

Creating a new product

// Use the class
use \RTL\Products\cProduct;

// Create an empty instance
$product = new cProduct();

// Create a new product
$status = $product->create($title = "iPhone");

Update information and remove products

Using the retail SDK you have full access to the products of the team, allowing to edit and remove products as well:

// Use the class
use \RTL\Products\cProduct;

// Create an existing product instance
$product = new cProduct("1_1");

// Update information
$product->update($title = "Apple iPhone");


// And if we want to remove the product, we simply write:
// This action cannot be undone
$product->remove();
