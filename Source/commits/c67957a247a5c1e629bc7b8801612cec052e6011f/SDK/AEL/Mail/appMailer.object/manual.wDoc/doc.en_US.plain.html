Application Mailer

Using this application mailer it is easy to send email using the mailgun API.

You can use this class to send mail in two modes:

  • - Application Mode
  • - Team Mode

Application Mode

When using application mode for the mailer, the from address will be have the following format:

Account Title <[username]@[application_name].apps.drov.io>

Team Mode

When using team mode for the mailer, the from address will be have the following format:

Account Title <[username]@[team_name].drov.io>

The class will use the username of the user to identify the sender but if there is no username set, it will use just the word "user".

// Use the class
use \AEL\Mail\appMailer;

// Create an appmailer instance in application mode
$appMail = new appMailer(appMailer::MODE_APP);

// Add recipient
$appMail->addRecipient("john.doe@example.com");

// Send the email
$response = $appMail->send("Mail Subject", "Text Mail Content!", "
HTML Mail Content!
");

Attachments

The mailgun API allows us to add attachments to the email. This class provides direct access to the team directory and you can attach a file directly by giving the inner path.

// Use the class
use \AEL\Mail\appMailer;

// Create an appmailer instance in application mode
$appMail = new appMailer(appMailer::MODE_APP);
$appMail->addRecipient("john.doe@example.com");

// Add attachment from the shared directory of the app
// This way you can attach files that are part of the team and the application only
$appMail->addAttachment($filePath = "/reports/report.pdf", $shared = FALSE);

// Send the email
$response = $appMail->send("Mail Subject", "Text Mail Content!", "
HTML Mail Content!
";
