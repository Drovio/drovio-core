Application resource reader

This class allows the application to read files from the resources folder. It can detect when in production environment and when in development environemnt and reads the files accordingly.

This class is for reading only files.

Example

// Use the class
use \AEL\Resources\resource;

// Read a file
$fileContents = resource::get("/templates/email_template.html");

// Send email
$mailer = new mailer();
