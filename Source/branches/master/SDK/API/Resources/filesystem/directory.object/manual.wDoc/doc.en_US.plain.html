System directory manager

Directory interface that provides the user functions to directory manipulation.

Functions of this class are accessible only through the SDK. Modules and other views cannot have access and create/remove/edit folders.

Example

// Use the class
use \API\Resources\filesystem\directory;

// Get all contents of a directory
$contents = directory::getContentList($directory = "/", $includeHidden = FALSE, $includeDotFolders = FALSE);

// The above call will create two arrays in the $contents like the following:
// $contents['dirs'] = array("test", "test1", "test2");
// $contents['files'] = array("log_file.txt");


// Get details about a directory/file, like getContentList() with more details
$details = directory::getContentDetails($directory = "/", $includeHidden = FALSE);

// The above will return an array of all the details:
// $details['files']['log_file.txt']['name'] = "test";
// $details['files']['log_file.txt']['extension'] = "txt";
// $details['files']['log_file.txt']['lastModified'] = "12240047979";
// $details['files']['log_file.txt']['size'] = "34406";
// $details['files']['log_file.txt']['type'] = "file";
