System folder manager

You can access any folder in the system using this folder manager.

Functions of this class are accessible only through the SDK. Modules and other views cannot have access and create/remove/edit folders.

Example

// Use the class
use \API\Resources\filesystem\folderManager;

// Create a folder in team mode
folderManager::create($path = "/logs/", $name = "day1", $recursive = TRUE);


// Remove a folder
// You cannot just remove folders. The folder must be either in account or in team
// You cannot remove system folders of any kind
folderManager::remove($path = "/logs/", $name = "day1", $recursive = TRUE);


// You can also clean a folder from its contents
// Same restrictions with removing folders
folderManager::clean($path = "/logs/", $name = "day1", $includeHidden = TRUE);
