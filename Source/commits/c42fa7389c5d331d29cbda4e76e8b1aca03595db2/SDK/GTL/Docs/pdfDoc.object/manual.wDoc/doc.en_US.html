PDF document handler with php

This class provides the right tools for creating a pdf document using php commands. This tool uses the tFPDF class which is an extension of the FPDF class.

For a complete manual of the commants, see the FPDF Manual

NOTE: This class supports by default the DejaVu font family which supports most of the UTF-8 fonts. No other fonts are currently supporter.

Simple example

The examples are similar to the FPDF tutorials, however we present a small functionality here:

// Use the class
use \GTL\Docs\pdfDoc;

// Create an instance
$pdf = new pdfDoc($orientation = 'P', $unit = 'mm', $size = 'A4');

// Add a new page
$pdf->AddPage();

// Write a new line
$pdf->Cell(40,10,'Hello World!');

// Get the pdf
$pdfContents = $pdf->Output();

// Save the pdf somewhere using a fileManager
...

Inserting images

Since the platform does not give direct access to files, this class has an interface which can put images into the pdf giving the image contents and providing the image type.

// Use the class
use \GTL\Docs\pdfDoc;

// Create an instance
$pdf = new pdfDoc($orientation = 'P', $unit = 'mm', $size = 'A4');

// Add a new page
$pdf->AddPage();

// Add an image from contents
$imageContents = ...
$pdf->ImageFromContents($imageContents, 10, 10);

// Get the pdf and save with a filemanager
$pdfContents = $pdf->Output();
...
