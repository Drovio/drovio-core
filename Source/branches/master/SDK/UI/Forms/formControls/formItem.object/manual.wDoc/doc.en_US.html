Platform Generic Form Item

Form builder/factory allows the creation of items with all the appropriate attributes in single-line calls. formItem is an internal object that is being used by the form builder in order to have less, more flexible and smarter code.

PHP Examples

// Single-line creator of a form item
	
// Use class
use \UI\Forms\formControls\formItem;

// Create a form item
$fi = new formItem();

// This is a UIObject, so we build
// We use the chain pattern to get the DOMElement immediately
$item = $fi->build($tag = "select", $name = "gender", $id = "", $value = "", $class = "", $context = "")->get();

// Append item to anything
DOM::append($anything, $item);

Javascript interface + examples

formItem also provides a javascript interface where a developer can create form items on the client side dynamically.

// Create a form item in a single line
// formItem.create(tag, name, id, value, itemClass);
var fi = formItem.create("input", "username", "item_id", "john", "item_class");

// Append to form
jq("form.userform").append(fi);
