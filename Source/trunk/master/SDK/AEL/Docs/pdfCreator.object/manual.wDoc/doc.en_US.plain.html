Application pdf creator

This class extends the pdfDoc class with all of its functions. It adds two new interfaces that allow to insert images directly from the team shared or private folder for the current application.

// Use the class
use \GTL\Docs\pdfDoc;

// Create an instance
$pdf = new pdfDoc($orientation = 'P', $unit = 'mm', $size = 'A4');

// Add a new page
$pdf->AddPage();

// Add an image from private team directory
$pdf->ImageFromTeam("/logo.png", 10, 10);

// Add an image from shared team directory
$pdf->ImageFromTeamShared("/logo2.png", 30, 20);

// Get the pdf and save with a filemanager
$pdfContents = $pdf->Output();
