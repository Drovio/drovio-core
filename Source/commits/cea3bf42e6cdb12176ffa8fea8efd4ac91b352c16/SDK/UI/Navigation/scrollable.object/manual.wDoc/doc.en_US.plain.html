Scrollable interface

Scrollable is a simple interface that makes the page scroll to a selected jQuery object on a smooth way.

// Scroll to a header
Function:	scrollHere()
Description:	Scroll to a selected jQuery element
Usage:		jqElement.scrollHere();

Example:
// Get an element to scroll to
var scrollHeader = jq("h1.important");
// Scroll to the element
scrollHeader.scrollHere();
