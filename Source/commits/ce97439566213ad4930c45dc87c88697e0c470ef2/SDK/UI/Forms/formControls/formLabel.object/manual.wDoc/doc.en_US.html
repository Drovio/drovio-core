Platform Generic Form Label

Creating a form label is extremely useful when we have to create a form with proper directions and labels. formLabel is an internal object which facilitates the process of creating labels along with inputs to build better and quicker forms.

PHP Examples

// Single-line creator of a form label
	
// Use class
use \UI\Forms\formControls\formLabel;

// Create a form label
$fl = new formLabel();

// This is a UIObject, so we build
// We use the chain pattern to get the DOMElement immediately
$label = $fl->build($context = "Label title", $for = "")->get();

// Append item to anything
DOM::append($anything, $label);

Javascript interface + examples

formLabel also provides a javascript interface where a developer can create form labels on the client side dynamically.

// Create a form label in a single line
// formLabel.create(context, forInputID);
var fl = formLabel.create("Label title", "");

// Append to form
jq("form.userform").append(fl);
