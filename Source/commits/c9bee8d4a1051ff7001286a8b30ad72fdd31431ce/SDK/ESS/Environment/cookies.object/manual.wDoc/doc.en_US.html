System Cookies manager

Manage website cookies through php and javascript. This class allows the developers to create and remove cookies accross the platform.

PHP Interface

Creating cookies on the backend is simple, writting only one line of code:

// Use the class
use \ESS\Environment\cookies;

// Create a new cookie for one day (expiration is in seconds)
// The cookie will be available on the next call
cookies::set($name = "cookie_name", $value = "cookie_value", $expiration = 60 * 60 * 24, $httpOnly = FALSE, $secure = FALSE);

// Get a cookie value
$cookieValue = cookes::get($name = "cookie2_name");

// To remove a cookie, just set a negative expiration value
// This will cause the browser to delete the cookie
cookies::set($name = "cookie_name", $value = "cookie_value", $expiration = -1, $httpOnly = FALSE, $secure = FALSE);

// Or call remove
cookies::remove($name = "cookie_name");

Javascript interface

We can also handle cookies through javascript. Javascript provides simple methods to handle cookies but we created a simple interface which we can use to read a d write platform cookies.

// Javascript interface
	
// Read a cookie
Function:	get()
Description:	Get a cookie value
Usage:		cookies.get(c_name);

// Set a cookie
Function:	set()
Description:	Set a cookie value. Provide expiration in days.
Usage:		cookies.set(c_name, value, exdays, path);
Example:	cookies.set("cookie_name", "cookie_value", 1, "/");

// Example to remove a cookie:
cookies.set("cookie_name", 1, -1, "/");

NOTE: Keep in mind the cookies restrictions based on the httpOnly and secure parameters.
