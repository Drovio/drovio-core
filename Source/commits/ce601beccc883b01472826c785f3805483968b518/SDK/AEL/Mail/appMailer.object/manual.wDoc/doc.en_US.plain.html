Application Mailer

Using this application mailer it is easy to send email using the mailgun API.

You can use this class to send mail in two modes:

  • - Application Mode
  • - Team Mode

Application Mode

When using application mode for the mailer, the from address will be have the following format:

Account Title <[username]@[application_name].apps.drov.io>

Team Mode

When using team mode for the mailer, the from address will be have the following format:

Account Title <[username]@[team_name].drov.io>

Example

// Use the class
use \AEL\Mail\appMailer;

// Create an appmailer instance in application mode
$appMail = new appMailer(appMailer::MODE_APP);

// Add recipient
$appMail->addRecipient("john.doe@example.com");

// Send the email
$response = $appMail->send("Mail Subject", "Mail Content!", appMailer::CONTENT_TXT);
