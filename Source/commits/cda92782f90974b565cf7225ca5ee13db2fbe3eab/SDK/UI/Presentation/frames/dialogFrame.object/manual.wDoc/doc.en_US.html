Window Dialog Frame

A dialog frame is an extended frame that includes an embedded form. A dialog frame shows a popup dialog to the user in order to make an action. The frame includes a form (simpleForm) where the developer can insert any type of input. Even simpleForm elements can be inserted.

// Create and show a simple dialog frame to the user
	
// Use the class
use \UI\Presentation\frames\dialogFrame; // Extends the windowFrame

// Create a frame
$frame = new dialogFrame($id = "");

// This is a UIObject, so we build
$frame->build($title = "Dialog Frame", $action = "/user/info/update.php", $background = TRUE, $type = self::TYPE_OK_CANCEL);

// The dialog includes a form factory (simpleForm template) which we can get and create more inputs
// We can append form elements in the form or in the frame directly, either way all the elements will be inside the form
$form = $frame->getFormFactory();


// Add a hidden input of the user id
$input = $form->getInput($type = "hidden", $name = "uid", $value = $userID, $class = "", $autofocus = FALSE, $required = FALSE);
$form->append($input);

// User name
// Build a form row with label and input
$input = $form->getInput($type = "text", $name = "username", $value = "", $class = "", $autofocus = FALSE, $required = TRUE);
$frow = $form->buildRow("Username", $input, $required = TRUE, $notes = "");
$form->append($frow);

// After we finish building, we return the frame report
// This will return a persistent popup
return $frame->getFrame($requiredFormNotes = TRUE);
