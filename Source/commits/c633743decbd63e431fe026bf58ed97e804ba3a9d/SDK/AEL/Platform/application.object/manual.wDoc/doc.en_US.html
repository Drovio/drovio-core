Application platform manager

Every application when is being executed has access to the platform application manager. This class allows developers mostly to import classes form the application source.

PHP Example

// In application source object
	
// Import other application classes
application::import($package = "Main", $class = "test/testObject");

// Use the object
use \APP\Main\test\testObject;

This is the master class from which APPContent gets access like loading views and managing loading depth of views.
