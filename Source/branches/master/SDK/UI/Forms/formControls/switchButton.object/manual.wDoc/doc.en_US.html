Interactive Switch Button

Switch button is a simple Drovio control that provides functionality to turn on and off switches. Instead of a normal checkbox, you can use this switch butto

Build a switch button and append to a form:

// Use the class
use \UI\Forms\formControls\switchButton;

// It is a UIObject, so we have to initialize and build
$sb = new switchButton();
$switchButton = $sb->build($active = FALSE, $name = "", $value = "", $class = "")->get();
DOM::append($form, $switchButton);

The above code will create a normal switch button. It can be appended to forms and act as a normal checkbox.
