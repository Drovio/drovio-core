Account profile manager

This class provides access to the account profile information. It works only for the current logged in account.

This class is part of the Profile API allowing the developer to read and edit the account information. It also provides access to account public information.

Validating current account

A simple use of this interface is when we want to check if there is a logged in account for the system and act accordingly:

// Use the class
use \API\Profile\account;

// Check if there is a valid account logged in
if (account::validate())
{
	// Get account id
	$accountID = account::getAccountID();
}
else
{
	// Show a login dialog popup
}

Getting account information

There are many times when we want to display account information to the user for listings or just display the account information:

// Use the class
use \API\Profile\account;

// Get account title and display
$accountTitle = account::getAccountTitle();

// Get account profile image
$accountInfo = account::info();
if (isset($accountInfo['profile_image_url']))
{
	// Create an image div
	$img = DOM::create("img");
	DOM::attr($img, "src", $accountInfo['profile_image_url']);
	DOM::append($ico, $img);
}
